# 少儿编程课程平台执行步骤

## 阶段一：基础架构搭建（4周）

### 第1周：项目初始化与环境搭建

#### 周一：前端项目初始化
1. **创建前端项目目录**
   ```powershell
   mkdir frontend
   cd frontend
   ```

2. **初始化React+TypeScript项目**
   ```powershell
   npx create-react-app . --template typescript
   ```

3. **安装核心依赖**
   ```powershell
   npm install @reduxjs/toolkit react-redux react-router-dom antd axios socket.io-client styled-components
   ```

4. **配置项目结构**
   - 创建 `src/components`、`src/pages`、`src/store`、`src/services` 等目录
   - 配置 `tsconfig.json` 和 `eslintrc.js`

#### 周二：后端服务搭建
1. **创建后端项目目录**
   ```powershell
   mkdir backend
   cd backend
   ```

2. **初始化Node.js项目**
   ```powershell
   npm init -y
   ```

3. **安装核心依赖**
   ```powershell
   npm install express mongoose jsonwebtoken redis multer aws-sdk cors helmet morgan
   ```

4. **安装开发依赖**
   ```powershell
   npm install --save-dev typescript @types/express @types/mongoose @types/jsonwebtoken @types/redis @types/multer @types/cors nodemon ts-node
   ```

5. **配置项目结构**
   - 创建 `src/routes`、`src/controllers`、`src/models`、`src/middleware`、`src/services` 等目录
   - 配置 `tsconfig.json`

#### 周三：数据库设计与初始化
1. **安装MongoDB**
   - 下载并安装MongoDB Community Edition
   - 启动MongoDB服务

2. **创建数据库**
   ```powershell
   mongo
   use scratch-platform
   ```

3. **创建初始集合**
   - users
   - teachers
   - students
   - courses
   - course_modules
   - lessons
   - projects
   - scratch_resources

4. **创建数据库连接配置**
   - 在后端项目中创建 `src/config/database.ts`
   - 配置MongoDB连接字符串

#### 周四：基础配置与工具搭建
1. **配置环境变量**
   - 创建 `.env` 文件
   - 配置数据库连接、JWT密钥、端口等

2. **搭建日志系统**
   - 配置Morgan日志中间件
   - 设置日志文件存储

3. **配置CORS和安全中间件**
   - 配置Helmet安全头
   - 配置CORS策略

#### 周五：项目结构优化与测试
1. **优化前端项目结构**
   - 配置路由结构
   - 搭建Redux store

2. **优化后端项目结构**
